Wags to Riches

Wags to Riches is designed to help connect adorable pets to loving families! This project pulls from the PetFinder API, returning pet data such as breed, location, name, and photo data. It uses Node.js, handlebars, MySQL, and express.js.

Employee Tracker

This application makes managing an employee database easy. View, add, update, and delete all departments, roles, employees, and budgets in one place! Requires usage of Inquirer, MySQL2, and express.js.

Beat Buddy

BeatBuddy is your friendly music hub! This is a web application that queries YouTube and MusicBrainz API and returns music videos and artist information. It uses JavaScript, jQuery, HTML, and CSS.

Ecommerce Back-End

This project is a CRUD app for an ecommerce site. It uses MySQL, express.js, sequelize, and dotenv.

SVG Generator

This project is a CLI utility to generate an SVG logo with shapes and text. It uses Node, Inquirer, HTML.

Note Taker

Note taker is a simple note taking CRUD app. It uses a simple JSON file as a database backed by a node server.
